# Hillkoff Asset Management Platform

Enterprise asset lifecycle platform foundation built with Next.js, TypeScript,
Tailwind CSS, shadcn/ui conventions, Firebase, BigQuery, Cloud Functions, and
Vercel.

## Commands

```bash
npm install
npm run dev
npm run check
npm run build
npm run functions:build
```
